## Text Editor

```
A tkinter based Text Editor in Python
```
Done in Early learning stages of python and tkinter

* File Mode : SAVE, OPEN, NEW, EXIT
* Edit Mode : CUT, COPY, PASTE
* Help Mode : HELLPPP
